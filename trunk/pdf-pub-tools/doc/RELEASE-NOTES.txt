== Tools ==

=== Command Line Interfaces (CLI) ===

==== TrainingBookPublisherCLI ====

net.mitnet.tools.pdf.book.publisher.ui.cli.TrainingBookPublisherCLI
purpose: creates final training PDF book
usage: TrainingBookPublisherCLI -v -i <input-dir> -o <output-dir> -b <output-pdf-book>

==== OpenOfficeDocConverterCLI ====

net.mitnet.tools.pdf.openoffice.ui.cli.OpenOfficeDocConverterCLI
purpose: converts ODT to PDF (by default)
usage: OpenOfficeDocConverterCLI -v -i <input-dir> -o <output-dir>

net.mitnet.tools.pdf.pdf.builder.ui.cli.PdfBookBuilderCLI
purpose: builds PDF book from PDF files
usage: PdfBookBuilderCLI -v -i <input-dir> -o -b <output-pdf-book>

=== GUI ===

==== TrainingBookPublisherGUI ====

net.mitnet.tools.pdf.book.publisher.ui.gui.TrainingBookPublisherGUI
purpose: creates PDFs from ODT only at this stage; a bit more work to make this equivalent to
TrainingBookPublisherCLI
usage: launch from Eclipse
